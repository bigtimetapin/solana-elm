# solana + elm-lang
Fullstack development on the solana blockchain with elm-lang

## Build
* typescript
```shell
cd ./web/js; npm run build
```

* css
```shell
cd ./web/sass; npm run build
```

* elm
```shell
./build.sh
```

* anchor
```shell
anchor build
```

* localhost
```shell
cd ./web http-server
```
